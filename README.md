# DIGIX Asset Ops -- Mobile

React Native (Expo) field technician app for the DIGIX Asset Management & Operations Platform.

## Stack

- React Native (Expo SDK 52), TypeScript, Expo Router
- expo-location (GPS tracking), expo-camera (photos), expo-secure-store

## Quick Start

```bash
npm install
npx expo start
```

## Features

- GPS check-in / check-out for attendance
- Assigned ticket management with photo attachments
- QR/barcode asset scanning
- Background GPS tracking for live location
- Pre/post installation photo capture
